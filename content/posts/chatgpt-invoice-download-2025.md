## 为什么很多人找不到 ChatGPT 发票？

### 1. 入口不明显
OpenAI 把账单藏在 Settings > Billing，不是显眼的“我的订单”。

### 2. 支付渠道不同，路径不同
- Web 直充 → OpenAI 官网出票  
- iOS App Store → Apple 发票  
- 代充平台 → 第三方凭证

### 3. 发票不是实时生成
扣款后需 1-3 小时 才出现在账单页，尤其是跨境支付。

## 四端下载发票全流程（带截图级指引）

### Web 版（最全，支持税号修改）
1. 登录 [chat.openai.com](https://chat.openai.com)  
2. 左下角头像 → Settings & Beta  
3. 左侧菜单 → Billing  
4. 点击 Payment history  
5. 找到当月记录 → Download invoice  
6. PDF 自动下载，含 OpenAI 公司抬头、税号、扣款明细

> 小技巧：点击 Edit tax info 可添加公司税号，用于报销

---

### iOS App（Apple 官方发票）
1. iPhone 设置 → Apple ID（顶部头像）  
2. 媒体与购买项目 → 查看账户  
3. 购买记录 → 找到 “OpenAI” 或 “ChatGPT Plus”  
4. 点击 → 总计已付 → 复制发票编号  
5. 打开 [reportaproblem.apple.com](https://reportaproblem.apple.com)  
6. 粘贴编号 → 下载 PDF（含 Apple 税号）

> 注意：Apple 发票抬头为个人，无法改企业名

---

### 安卓 App（Google Play 发票）
1. 打开 Google Play → 右上角头像 → 付款和订阅  
2. 预算与订单记录 → 找到 “ChatGPT”  
3. 点击订单 → 下载发票 或发送邮件  
4. 收到带 Google 抬头的 PDF

> 若用礼品卡充值，显示“Google Play 余额扣款”

---

### API 用户（企业批量导出）
1. 登录 [platform.openai.com](https://platform.openai.com)  
2. 左侧 Billing → Usage & Invoices  
3. 选择月份 → Export CSV 或 Download PDF  
4. 支持批量，含项目 ID、用量、单价

## 报销必备：发票模板与税务要点

### 标准报销模板（Word 可复制）
```
发票抬头：OpenAI, Inc.
发票号码：INV-2025-11-XXXX
开票日期：2025-11-01
金额：USD 20.00（含税）
服务内容：ChatGPT Plus 订阅服务（AI 软件服务费）
税号：XX-XXXXXXX（已填）
```

### 税务提醒
- 中国企业：可作为“技术服务费”入账，需提供外汇支付凭证  
- 个税抵扣：暂不支持，属于消费性支出  
- 增值税发票：OpenAI 不开具，需走“境外服务”流程

## FAQ：发票下载常见问题全解

为什么邮箱没收到发票？ 
OpenAI 不自动发邮件，必须去 Billing 页面手动下载。

代充平台能出 OpenAI 官方发票吗？ 
不能。代充只给收据，企业报销需登录 Web 版补开 OpenAI 发票。

iOS 发票能改成公司抬头吗？ 
不行。Apple 抬头固定为个人名，企业报销必须用 Web 版。

扣款当天能下载发票吗？ 
需等 Stripe 结算，最快 1 小时，最慢次日。

历史账单能补开吗？ 
Billing 页面可查近 24 个月记录，随时下载。
