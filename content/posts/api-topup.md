## 风控要点
- 保持登录地区与账单地区一致  
- API Key 不要在客户端硬编码泄露

## 充值方式
- 官方绑卡自动扣费  
- 第三方信用点数代充（需评估合规）

## 账单管理
- 定期下载发票  
- 设置消费提醒与配额上限
