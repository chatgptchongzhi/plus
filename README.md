# 木子AI (plus)

> 一个可直接部署到 GitHub Pages 的纯前端网站。
> 模仿 pinzhanghao.com，主打 ChatGPT / Sora / AI 工具教程与充值引导。

## 功能特性
- 自动读取 Markdown 文章生成首页
- 推荐区 + 单列文章 + 翻页 + 标签筛选
- 右侧栏固定 + 悬浮二维码 + 阅读量统计
- 自动生成目录 (TOC) + SEO 优化
- 内置 Decap CMS 后台，无需服务器

## 部署方法
1. 上传所有文件到 GitHub 仓库 `plus`
2. 在 Settings → Pages → 选择 `main / root`
3. 打开 `https://你的用户名.github.io/plus/`
